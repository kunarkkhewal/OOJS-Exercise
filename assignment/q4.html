Properties of Argument

1. arguments is an array-like object that contains the values of all the arguments passed.

2. arguments is similar to array, that is it have length property and index starts from 0 but other array properties like push() and pop() is not available for arguments.

3. arguments is not available in arrow function, in arrow function it gives arguments not defined reference error

arguments.callee : callee is a property of the arguments object. It can be used to refer to the currently executing function inside the function body of that function.

arguments[@@iterator] : It returns a new Array iterator object that contains the values for each index in the arguments.

arguments.length: The number of arguments that were passed to the function.
